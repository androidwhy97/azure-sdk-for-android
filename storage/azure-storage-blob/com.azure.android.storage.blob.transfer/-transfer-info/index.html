TransferInfo

Information about a particular transfer identified bygetId.

Information include the currentState, progress of the transfer.The state can be retrieved by callinggetState.The progress can be retrieved by callinggetProgress.

Note that progress is only available for the state (RECEIVED_PROGRESS,getProgress() returns{@code null}for all other states.
