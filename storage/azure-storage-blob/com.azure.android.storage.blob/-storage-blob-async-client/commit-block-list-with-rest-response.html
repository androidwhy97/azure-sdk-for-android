commitBlockListWithRestResponse

The Commit Block List operation writes a blob by specifying the list of block IDs that make up the blob.For a block to be written as part of a blob, the block must have been successfully written to the server in a priorstageBlockoperation. You can callcommit Block List to update a blob by uploading only those blocks that have changed, then committing the new and existingblocks together. You can do this by specifying whether to commit a block from the committed block list or fromthe uncommitted block list, or to commit the most recently uploaded version of the block, whichever list it may belong to.
