Creates aBuilderthat uses the providedcom.azure.android.core.http.ServiceClient.Builderto build aServiceClientfor theStorageBlobAsyncClient.

The builder producedServiceClientis used by theStorageBlobAsyncClientto make Rest API calls.MultipleStorageBlobAsyncClientinstances can share the sameServiceClientinstance, for e.g.when a newStorageBlobAsyncClientis created from an existingStorageBlobAsyncClientthroughnewBuilder()}then both shares the sameServiceClient.TheServiceClientcomposes HttpClient, HTTP settings (such as connection timeout, interceptors)and Retrofit for Rest calls.
